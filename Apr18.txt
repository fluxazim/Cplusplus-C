Terminal commands:
1. ls : "List" the contents of the directory or folder
2. ls -l : shows long listing format. shows permission, size, etc
3. ls -a : shows all files including the hidden ones including the dots
4. cd : change directory 
5. (cd ../) : to go back one place in directory
